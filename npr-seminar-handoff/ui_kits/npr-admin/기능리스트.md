## 구(v2.0) 명세 대비 구현 영향 요약 (npr-seminar 참고)

- 역할 3종(owner/siljang/gangsa) → **단일 관리자** — 역할 게이팅·ROLE_MODULES 불필요.
- **전화예약(phone) 모듈 삭제** — 수동 예약은 예약 명단 드랍다운·수동 추가로 흡수.
- 상담(counsel 슬롯·신청) → **간담회 placeholder** — CounselSlot/CounselBooking 도메인 보류.
- **캠퍼스 3종 + 단위(unitOf) 체계 신설**, Student에 학번·모/부 연락처·담임명 추가.
- Reservation에 reservedBy·source·scannerNo·logs[] 추가, 리마인드(Reminder) 삭제 → **만족도 설문(SurveyResponse 확장: rating·comment·photo)**.
- 통계 시간대 라인·전환율 삭제 → 단위별 예약률·참석률.
- 엑셀 업/다운로드(재원생) 삭제 → 설문 결과 엑셀 저장(목업)만.
